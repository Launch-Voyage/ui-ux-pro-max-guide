# UI UX Pro Max — 161개 업종별 디자인 룰 (한국어 해설판)

161개 업종·개인 앱 유형별 UI/UX 디자인 룰을 **'왜 이런 룰인지'까지** 풀어낸 입문자용 레퍼런스.

**Live**: https://ui-ux-pro-max-guide.vercel.app

## 구성

- **용어 사전 20종** — 글래스모피즘, 뉴모피즘, 브루탈리즘 등 스타일 용어를 시각 예시와 함께 해설
- **6가지 관통 원칙** — 161개 룰에서 반복되는 핵심 패턴 요약
- **161개 룰 카드** — 업종별(1–90) + 개인 앱(91–161). 각 카드는 왜 / 패턴 / 스타일 / 컬러 / 타이포 / 효과 / 6색 팔레트 / 필수·조건 / 금지 항목으로 구성
- 검색(업종명·스타일·키워드·번호)과 그룹 필터 지원

## 구조

정적 페이지 — 빌드 없음. `index.html`(마크업 + 렌더링 JS) + `data.js`(룰 데이터). 로컬에서는 `index.html`을 브라우저로 열면 바로 동작합니다.

## 출처와 라이선스

[nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) (MIT License)의 `ui-reasoning.csv`(161개 룰)와 `colors.csv`(팔레트) 데이터를 한국어로 번역하고 해설을 더한 것입니다. 용어 사전의 시각 예시와 '왜' 해설은 이 저장소에서 새로 작성했습니다.
